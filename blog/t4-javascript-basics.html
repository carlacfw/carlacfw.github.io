  • Control flow and loops, what are they?

    • Control flow is used to make decisions into the JS, we use the if/else statements to control this decisions. A daily example of control flow is if it is sunny I won't wear my rain coat. But (else) if it is rainy I need to wear my rain coat. So we can understunding it as decisions made by consequences.

    Loops is when the programmer wants to do something for a fixed number of times, so it will be writing to do the same over and over again, no need to be written every time. Comparing with an alarm, you can set it to go off Monday-Friday at 6:30AM, then it will work and you won't need to set the alarm every single day. This is how a loop works too.

  • The difference between accessing data from arrays and objects.

    • Arrays are written inside brackets and its index, which is a numerical string, is based in zero which means that de data stored in it beggins at zero. Objects are something else and contains a collections of properties and we can add or remove this properties as we want, opposite the arrays.

    So the difference between both to acces their data, is that the arrays can only be accessed by brackets while the objects can be accessed in two ways, using dot objectName.propertyName or brackets objectName["propertyName"].

  • What functions are and why they are useful.

    • Function is a block of code made to execute a particular task. It can be used in many times and also no need to write a code over again. All you need to do is call the function and execute the function you want. For exemple function myFunction(p1, p2) { return p1 * p2; } Where function is a keyword to call itself.
