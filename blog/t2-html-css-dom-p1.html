HTML, CSS and DOM What are they?


  • Describe HTML, CSS and DOM to a no-technical person:

    • HTML is a language used to write a web page. CSS makes the HTML looks pretty on the web page (backgroung color, font size, font style is all CSS funcionalities) and DOM makes the browser read all this informations we wrote in HTML and CSS then show it on the web page.

  • What is boxifing desing?

    • You can imagine all the elements of a web page, inside a big invisible box and around these elements has another invisible box, for exemple, imagine a paragraph, around this paragraph has a box. So boxifing is it, divide the elements of a web page into invisible boxes, try to draw it, it can be easier to understand.

  • What is the box model?

    • Box model is a term used to describe the layout of a web page.
