Responsive Web Site, Design, Framework, Wireframe...What are they?

Explaining to a non-technical person...

  • What a responsive site is, and why responsiveness is important?

    • A responsive page is a page which the size changes to fit in any screen size device you are accessing it.

  • What mobile first design is, and why it's important?

    • The mobile design is the layout of a web page made to be seen on the screen of your mobile phone. It's been now the first option of design to create a web page because most of the users access a web page via mobile.

  • What frameworks are, and their pros and cons?

    • Frameworks are a common structure to build a web site or a programme, but let's talk about web site. You can use a framework to make differents web pages, we can try to compare with a CV template, you have this template but you can build your CV in different ways acording to the job application you are working in and so the Frameworks.   • What a wireframe is and why we use it?

    • A wireframe is like a draw of the web page, where we are going to place any element of our page. We use it to planning the layout of the web page before built it.
