Arrays

An array is a data structure that contains a group of elements.

The functions .map() .filter() .reduce() running using a function as a parameter through objects of an array

The .filter() is for filter the objects of an array, selecting only the ones that you want. As an exemple, imagine that you have an array like this [1,2,3,4,5,6] and you want to select only the numbers that can be divided by 2, using the .filter() function you can get that.
The .map() is for transforming the objects of an array from one type to another type. If I have an array with number, using the .map() function it can turn into text.
The .reduce() is for converting an array in only one value. For exemple, one array with numbers, using the .reduce() function can sum those number to have the result. Reducing the array in one object.
