Scope in JavaScript

What 'scope' is, and how it works in JavaScript

Let's say that scope is the whole context inside a JavaScript file. We can define them as Global Scope and Local Scope.

Global Scope is this whole context, where we can declare variables, for example. And everything declared into the Gobal Scope can be accessed for the others variables or functios declared inside it.

Local Scope is declared after Global Scope, the easiest way to recognise it is because it is decalred inside curly braces. Everything declared inside the curly braces are locally and cannot access or be accessed for/what is into the Global Scope. It has it's own functions and variables and whatever you want to declare inside it and it will works only there, it's an specific action for an specific function.
